DmFlow中建立需求辭典：會議室和團隊人員

透過智能對話技術，輕鬆預約會議室，查詢可用時間，自動發送邀請給團隊成員。提升工作效率，實現順暢的團隊溝通與協作。

點標[1]點選詞典，點選下圖點標[2]新建新的自定義辭典，分別為"會議室"以及"團隊人員"。 會議室那一列點按鈕"詳情"，新增"103A"、"103B"、"104B"，也可以新增其他的名稱，如下圖。 團隊人員那一列點按鈕"詳情"，新增"小明"、"阿華"、"小花A"、"小花B"等等姓名，如下圖。
